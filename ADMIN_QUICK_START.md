# Admin Panel - Quick Start Guide

## Accessing the Admin Panel

Navigate to your admin area at:
```
http://yoursite.com/admin/
```

## Login
Use your admin credentials to access the dashboard.

## Dashboard Overview

The new dashboard displays:
- **Metric Cards**: Quick stats on portfolio items, about section, system status
- **Quick Actions**: Three main action cards to access key features
- **Recent Activity**: Timeline of system updates and status

## Navigation

### Left Sidebar (Desktop)
Available on all desktop and tablet screens:
- **Dashboard** - Main admin overview (home icon)
- **Edit About** - Manage profile section
- **Portfolio** - Create and manage portfolio items
- **All Content** - Unified content manager
- **Settings** - Account configuration
- **Change Password** - Update password
- **Logout** - Exit admin panel

### Mobile Menu
On mobile devices:
1. Tap the floating blue button at bottom-right to open sidebar
2. Select your option
3. Sidebar automatically closes

### Top Bar
Available on all screen sizes:
- **Search Input**: Search for content (coming soon)
- **Dark Mode**: Toggle dark theme (moon icon)
- **Notifications**: View system alerts (bell icon)
- **Profile**: View account info and quick access

## Key Features

### Portfolio Management
- Add new portfolio items with images
- Edit existing items
- Delete items
- Reorder items
- File upload with preview

### About Section
- Update heading and bio
- Change profile image
- Responsive image preview

### Content Manager
- Unified interface for all content
- Inline editing
- Quick updates

## Responsive Design

### Desktop (1024px and above)
- Sidebar always visible
- Full-width content area
- All features accessible

### Tablet (768px - 1023px)
- Sidebar visible
- Content adjusts width
- Touch-friendly buttons

### Mobile (below 768px)
- Sidebar hidden by default
- Floating menu button
- Single-column layout
- Touch-optimized interface

## Tips & Tricks

1. **Active Menu Indicator**: Your current page is highlighted in blue in the sidebar
2. **Quick Navigation**: Click any sidebar item to jump between sections
3. **Responsive Images**: Images automatically scale to fit your screen
4. **File Uploads**: Click the image upload button to select files from your device
5. **Mobile Access**: Full functionality available on mobile via hamburger menu

## Troubleshooting

### Sidebar Not Showing on Mobile
- Tap the blue floating button at bottom-right
- Check your browser zoom level (should be 100%)

### Images Not Displaying
- Ensure files are in supported formats (JPG, PNG, GIF, WebP)
- Check file size (recommended under 5MB)
- Verify file was uploaded successfully in the preview

### Search Not Working
- Feature coming soon - currently for UI display

## System Requirements

- PHP 7.0 or higher
- MySQLi database support
- Modern browser with CSS3 support
- JavaScript enabled
